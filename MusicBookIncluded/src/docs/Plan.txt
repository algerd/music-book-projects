1) Окна:
    - верхнее - меню
    - слева - дерево группы (Explorer)
    - справа - окно контента

2) Окна контента:
    1. Детальная информация (Details):
        - группы - доступно из Explorer, Overview групп, контентных окон (альбомов, песен, музыкантов)
        - альбома - доступно из Explorer, Overview альбомов, контентных окон (артистов, песен, музыкантов)
        - песни - доступно из Explorer, Overview песен, контентных окон (альбомов, песен, музыкантов)
        - музыканта - доступно из Overview музыкантов, контентных окон (артистов, альбомов, песен)

    2. Общая информация (Overview):
        - групп - доступно из Menu
        - альбомов - доступно из Menu
        - песен - доступно из Menu
        - музыкантов - доступно из Menu
      Содержит фильтры и поиски.    

3) Дерево группы (Explorer):
    группа->альбомы->песни
