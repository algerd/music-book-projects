
1) Порядок загрузки данных из бд.
    Никаких джойнов, они не соотвтствуют иерархии(дереву) классов.
    Загрузка данных должна идти в порядке ствол-ветвь-листья (от родителя к детям).
    Дерево musicbook:
       Artist{
            Albom1{
                Song1{}
                Song2{}    
                ...
            }
            Albom2{}
            ...
       }

    Песня может не иметь альбома или он неизвестен, тогда при добавлении такой песни создавать
    у группы альбом 'unknown' или 'singles' и добавлять такие песни туда. 
   
    1. Загрузить все сущности из 'artist' в таблицу ArtistTable в массив artists.
        У сущности artist создать 

    2. Загружать данные из 'album' в таблицу AlbumTable в массив albums.
        При создании сущности Album искать её родителя сущность Artists по полю id_artist
        и заносить эту сущность Album в массив родителя artists сущности Artist.

    3. Загружать данные из 'song' в таблицу SongTable в массив songs.
        При создании сущности Song искать её родителя сущность Album по полю id_album
        и заносить эту сущность Song в массив родителя albums сущности Album.
